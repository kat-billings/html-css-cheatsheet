HTML Table Reference

Elements
Tag Name Description
<table> Table Creates a table.
<tr> Table Row Creates a row. Nested inside the table element.
<td> Table Data Creates a cell of data. Nested inside the tr element.
<th> Table Head Nested inside the tr element
Organizational Elements
Tag Name Description
<thead> Table Header Nest the first row(s) of a table into this, if the row has headings in it.
<tbody> Table Body Nest the body of the table into this.
<tfoot> Table Footer Nest the last row of a table in this, if the table needs a footer. Used for stuff like totals.
Attributes
Attribute Name Description
scope Scope Lables th elements as being for rows or columns. Can be set to row or col.
rowspan Row Span Set to a number and the cell will be that many rows tall.
colspan Column Span Set to a number and the cell will be that many columns wide.
